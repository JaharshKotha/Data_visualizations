Findings

  • Array, string, method, arraylist, class are the top used words for an average user as depicted by the bubble cloud visualization. This visualization can be used to understand the areas of user interest for specific users or in general. This data can be used to structure the website and tailor it as per user interest

Visualization features

  • The bubble cloud gives the top words(top 30) where the size of the bubble indicates the frequency of use.
  • The user drop down gives an option to select the user to view their bubble cloud.

Data used

  • The words used for bubble cloud visualization have been extracted by parsing the URLs present in the class_operations dataset
  • Top 30 words for every user is used for the visualization
